Using webtext.io to make your website

IMPORTANT: Be wary of spaces in the syntax to ensure it is formatted correctly!

Here is the basic syntax on how to stylize your page:

Headers

  • Use "# " in front of text for Header 1
  • Use "## " in front of text for Header 2
  • Use "### " in front of text for Header 3
    • Example: "# Hello" will make Hello a Header 1.

Bold

  • Use "**" in front and after text for Bold
    • Example: "**Hello**" will make Hello Bold.

Italic

  • Use "*" in front and after text for Italic
    • Example: "*Hello*" will make Hello Italic.

Blockquote

  • Use ">" in front of text for Blockquote
    • Example: ">Hello" will make a Hello Blockquote.

Ordered Lists

  • Use "1. " in front of text to start an Ordered List. Following items must also have a number, but this can be any number.
    • Example:
      "1. Hello
       2. Hi"
      will make an Ordered List in that order.

Unordered Lists

  • Use "- " in front of text to start an Unordered List.
    • Example:
      "- Hello
       - Hi"
      will make an Unordered List in that order.

Code

  • Use "'" in front and after text for Code
    • Example: "'Hello'" will make a <code>Hello</code> Code tag.

Horizontal Rule

  • Use either "***", "___", or "---" in a line by itself for a Horizontal Rule
    • Example: "***" will make a Horizontal Rule.

Links

  • Use "[LINK TITLE](https://LINK)" to create a named Link
    • Example: "Go to [Example](https://example.com)" will make a Link named Example to example.com.

Images

  • Use "![ALT TEXT](IMAGENAME.jpg)" to create an Image with alt text
    • Example: "![Sunflowers](sunflowers.jpg)" will make an Image that you choose with your desired alt text.
